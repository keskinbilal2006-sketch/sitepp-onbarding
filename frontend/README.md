# Frontend (Web) - Klasor Aciklamasi

- `src/app`: Next.js App Router sayfalari ve layout yapisi
- `src/features`: Feature-based modül yapisi (api/hooks/components)
- `src/components/ui`: Ortak UI bilesenleri
- `src/lib`: API client, permission ve format helperlari
- `src/hooks`: Ortak custom hook giriş noktasi
- `src/types`: Ortak tipler
- `src/styles`: Global stiller
- `public`: Statik dosyalar
